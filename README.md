Exercícios de Condicionais em Java
**Exercício 1: Ler um valor e escrever a mensagem É MAIOR QUE 10! se o valor lido for maior que 10, caso contrário escrever NÃO É MAIOR QUE 10!**

Saída Esperada:
Se o valor lido for 15, a saída será: É MAIOR QUE 10!

Se o valor lido for 8, a saída será: NÃO É MAIOR QUE 10!

**Exercício 2: Ler um valor e escrever se é positivo ou negativo (considere o valor zero como positivo).**
            
Saída Esperada:

Se o valor lido for 7, a saída será: O valor é positivo.

Se o valor lido for -3, a saída será: O valor é negativo.

**Exercício 3: Ler dois valores (considere que não serão lidos valores iguais) e escrever o maior deles.**
            
Saída Esperada:

Se os valores lidos forem 12 e 8, a saída será: O maior valor é: 12

Se os valores lidos forem 5 e 15, a saída será: O maior valor é: 15

**Exercício 4: Ler um número e verificar se é par ou impar.**
            
Saída Esperada:

Se o número lido for 4, a saída será: O número é par.

Se o número lido for 7, a saída será: O número é ímpar.

**Exercício 5: Ler um número e verificar se está no intervalo entre 1 e 10.**
            
Saída Esperada:

Se o número lido for 7, a saída será: O número está entre 1 e 10.

Se o número lido for 15, a saída será: O número não está entre 1 e 10.

**Exercício 6: Ler um número e verificar se é maior ou menor que 18.**
            
Saída Esperada:

Se a idade lida for 16, a saída será: A pessoa é menor de idade.

Se a idade lida for 20, a saída será: A pessoa é maior de idade.

**Exercício 7: Ler um número e verificar se é maior que 100.**
            
Saída Esperada:

Se o valor da compra for 120, a saída será: Você tem direito ao desconto!

Se o valor da compra for 80, a saída será: Você não tem direito ao desconto.

**Exercício 8: Ler um número e verificar se é maior que 7.**
            
Saída Esperada:

Se a nota lida for 8.5, a saída será: Aluno aprovado.

Se a nota lida for 5.5, a saída será: Aluno reprovado.
